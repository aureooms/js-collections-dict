[js-collections-dict](http://aureooms.github.io/js-collections-dict)
==

Python's dictionnary data structure for JavaScript. Parent is
[js-collections](https://github.com/aureooms/js-collections).

```js
dict( ).getdefault( "x" , -2 ) ; // -2
```
